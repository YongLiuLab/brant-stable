Help information for Draw ROI:

type: draw roi as sphere or cubic
radius unit: unit of input radius
radius: radius of sphere or 1/2 edge length for cubic
input type: manual: input coordinates in coordinates (mm), e.g. 25,5,10; file: input a text file with 3 columns for x,y,z
mask roi afterwards: mask generated roi using input mask
output to one roi file: instead of output one file for each roi, brat will generate one roi file with each roi labeled by number
ref&mask: reference and mask file. for extracting information of origin, voxel size, bounding box, etc..
output path: directory for output

end
