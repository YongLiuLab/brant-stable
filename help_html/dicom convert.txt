Dicom Convert
	================================
	parallel workers: select how many workers to start a parallel work. The default is 0.
	convert to 4d: select to convert data to 4D format, otherwise to 3D.
	check timepoint: check the number of timepoints if filled with a positive number.
	del first timepoints: number of the timepoints to be deleted, the default is 10.
	name position: subject identifier for input data
	input dirs: directories can be input either using a .txt file or spm select window.
	output path: an output directory for saving results.
